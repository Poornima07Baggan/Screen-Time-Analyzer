# Screen Time Analyzer

Screen Time Analyzer is a Python Flask web application that analyzes daily screen usage across devices like mobile, laptop, and TV.

## Features
- Calculate total screen time
- Show screen time status (Normal or High)
- Display graph of screen usage

## Technologies Used
- Python
- Flask
- HTML
- Matplotlib
